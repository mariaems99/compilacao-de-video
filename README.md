📺 Página de Compilação de Vídeos
Este projeto é uma página HTML que reúne vídeos musicais selecionados, organizados em seções bem estruturadas para facilitar a navegação e manter a semântica correta.

🎯 Objetivo
Criar uma página simples em HTML que:
- Apresente um título principal (<h1>).
- Tenha um parágrafo introdutório logo abaixo do título.
- Contenha três seções (<section>), cada uma com:
- Um título (<h2>).
- Um parágrafo (<p>) introduzindo o vídeo.
- Um iframe (<iframe>) incorporando o vídeo.

📂 Estrutura do Código
- <main>: contém todo o conteúdo principal da página.
- <h1>: título da página.
- <p>: introdução geral.
- <section>: cada bloco de vídeo segue a ordem:
- <h2> → título da seção.
- <p> → introdução do vídeo.
- <iframe> → vídeo incorporado com atributos src, title, width, height.

✅ Requisitos atendidos
- Parágrafo introdutório logo abaixo do <h1>.
- Três <section> abaixo do primeiro <p>.
- Cada <section> começa com <h2>.
- Cada <section> contém <p> como segundo filho.
- Cada <section> contém <iframe> como terceiro filho.
- Todos os <iframe> possuem title e width.

🚀 Como usar
- Abra o arquivo index.html em qualquer navegador moderno.
- A página exibirá três seções com vídeos musicais:
- SZA - Broken Clocks
- Alcione - Você me vira a cabeça
- Beyoncé - Bigger
